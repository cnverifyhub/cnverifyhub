# Growth Marketing Playbook (Antigravity Skill)

Converting visitors into buyers is the only goal.

## 1. Conversion Mapping
- Attach event listeners to all primary "立即购买" (Buy Now) and "查看详情" (View Details) buttons.
- Log these events to a lightweight handler (Google Analytics, Baidu Tongji, or a custom Supabase logs table).

## 2. Funnel Analysis
- Track "Cart Addition" vs. "Final Checkout" ratios.
- If the drop-off is high at the payment screen, identify if the TXID input UI is confusing and suggest improvements.

## 3. SEO-Driven Content
- Periodically suggest adding new category descriptions or blog posts focused on trending keywords (e.g., "微信防封技巧", "2026年支付宝实名方案").
- Use these keywords to capture organic search traffic from Baidu.
