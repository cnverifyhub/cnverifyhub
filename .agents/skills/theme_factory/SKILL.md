# 🎨 Theme Factory (Antigravity Skill)

This skill enables the agent to apply consistent, high-end visual styles to project artifacts, documentation, and mockups. When invoked, present the following options and ask the user for their preference:

## 🌑 1. Dark Mode Elite (The "Premium" Theme)
- **Primary**: `#0f172a` (Slate 900)
- **Accent**: `#38bdf8` (Sky 400)
- **Typography**: Inter / JetBrains Mono
- **Aesthetic**: Sleek, technical, ultra-modern developer tool vibes.

## 🌈 2. Vibrant Discovery (The "Taobao-Plus" Theme)
- **Primary**: `#FF0036` (Taobao Red)
- **Accent**: `#FFD700` (Gold) / `#1677ff` (Alipay Blue)
- **Typography**: PingFang SC / Source Han Sans
- **Aesthetic**: High energy, trust-based, optimized for Chinese e-commerce conversion.

## 💼 3. Corporate Professional (The "Enterprise" Theme)
- **Primary**: `#1e293b` (Slate 800)
- **Accent**: `#2563eb` (Blue 600)
- **Typography**: Roboto / Open Sans
- **Aesthetic**: Safe, clean, standard corporate professionalism.

## ⚡ 4. Cyberpunk Neon (The "Aggressive" Theme)
- **Primary**: `#000000` (Pitch Black)
- **Accent**: `#f0abfc` (Fuchsia 300) / `#22d3ee` (Cyan 400)
- **Typography**: Space Mono / Orbitron
- **Aesthetic**: High contrast, dark-utility, futuristic hacker aesthetics.

---

> [!TIP]
> **Application**: Once a theme is chosen, all future `implementation_plan.md`, `walkthrough.md`, and `analysis_results.md` artifacts should use the terminology and color references from that theme.
